# Informed Governance Portal

## Launch the app

```bash
$ poetry install
$ poetry shell
$ export FLASK_APP=runserver.py
$ flask --debug run
```

```bash
$ curl http://127.0.0.1:5000/about/test
{
  "status": "OK"
}
```
